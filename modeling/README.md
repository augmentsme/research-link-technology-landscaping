# Overview

Our objective is to identify emerging trends of keywords. We do this by first obtaining grants data. Since the RLA api does not supply date information. We first get all nodes tagged with "Grant" tag from the neo4j database. This data contains all necessary info except the summary. So we first produce the `grants_raw.json` by querying the neo4j server, then for all ids, find the grants_summary field using RLA api. This process produce `grants_enriched.json` file. A sample json looks like
```json
{
  "funder": "National Health and Medical Research Council",
  "last_updated": "2016-05-16T10:00:32Z",
  "local_id": "1047630",
  "start_year": "2013",
  "nhmrc_funding_amount": "415218.3",
  "purl": "http://purl.org/au-research/grants/nhmrc/1047630",
  "source": "nhmrc.org",
  "type": "grant",
  "title": "Investigation of haematopoietic and leukemia stem cell self-renewal.",
  "end_year": "2016",
  "participant_list": "Dr Stephen Ting",
  "key": "nhmrc/1047630",
  "grant_summary": "The blood stem cell properties of self-renewal and multipotency allows for the constant replenishment of all blood components. Blood cancer stem cells use self-renewal to propagate disease, and can enter a quiescent-dormant phase to evade treatment. My research focuses on the identification and mechanisms of new genes that govern these unique blood stem cell properties, and to investigate whether these genes are also important in blood cancers."
}
```

I then clean the grants file by first filter out everything that are not nhmrc and arc grants and only keeps fields that are useful. 
NOTE: I should have also filtered out all grants that contains empty grant summary but it didnt matter that much

The cleaned grants have the following schema 

```json
{
  "id": "arc/DP220100606",
  "title": "Australia's variable rainfall - how dry or wet can it really get?",
  "grant_summary": "Australia's variable rainfall - how dry or wet can it really get? Australia’s rainfall is extremely variable, which means existing weather records are too short to calculate the true risk posed by droughts and floods. This project aims to quantify how naturally variable the rainfall coming from the Indo-Pacific mid-latitudes is, allowing recent rainfall extremes and future projections to be assessed in a long-term context. This project expects to produce new estimates of atmospheric moisture budgets between Australia and Antarctica based on a novel, 1000-year length reconstruction of moisture-bearing southern Indian Ocean storms. This new information is critically needed by water managers so that they can properly calculate (and ultimately prepare for) the worst of Australia’s rainfall-related risks.",

```

We then run extract workflow to extract 5 different types of keywords, `general`, 'problem', `application`, `technology` and `methodology`. 

the resulting keywords look like 

```json
{
  "term": "animal ethics",
  "type": "Methodology",
  "description": "A framework for evaluating the moral and societal implications of using animals in research and teaching.",
  "grants": [
    "arc/ANZCCART"
  ]
}
```
